# kralj.info

This repository contains the source code for my personal website, [kralj.info](https://kralj.info). The site serves as a portfolio to highlight my skills, experience, and contact information. It is a static website hosted on Firebase Hosting.

## Deployment

1. Install Firebase CLI (if not already installed):
   ```bash
   npm install -g firebase-tools
   ```

2. Login to Firebase:
   ```bash
   firebase login
   ```

3. Deploy to Firebase Hosting:
   ```bash
   firebase deploy
   ```
